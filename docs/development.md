# Development

TokenStack changes should be committed at coherent milestones using the Lore Commit Protocol from the workspace instructions.

Recommended stage commits:

1. Repository and tooling foundation.
2. Safety guard.
3. SQLite schema and local importer.
4. Read-only connectors.
5. Analytics and timezone.
6. Frontend data layer.
7. Command Center UI and accessibility.
8. Packaging and CI.
9. Documentation and screenshots.
